必须安装的软件：

nodejs官方下载地址 http://nodejs.cn/download/
win7系统最高支持node-v13.14.0-x64

git官方下载地址 https://git-scm.com
写博客软件官方下载地址 https://typora.io

环境搭建流程

  1. 安装Node.js淘宝镜像加速器（cnpm）
    npm install -g cnpm --registry=https://registry.npm.taobao.org

hexo博客官方下载网站 https://hexo.io/zh-cn/
hexo博客按照官网说明进行安装

  1. 全局安装hexo-cli客户端
    npm install hexo-cli -g
    如果安装失败卡住尝试用cnmp加速： cnpm install hexo-cli -g
    https://github.com/hexojs/hexo-starter //hexo-cli下载地址
    hexo -v 查看hexo版本

  2. 初始化目录
    创建博客安装目录，“blog”为文件夹名字，可以自定义名称
    hexo init blog

  3. 进入安装目录
    cd blog

  4. 安装博客系统
    npm install
    或者 cnpm install
    开启服务
    hexo server

  5. Git 初始化仓库信息:
    git config --global user.name "xxx"
    git config --global user.email "xxx@xx.com"

  6. 密钥管理
    ssh-keygen -t rsa -C "xxx@xx.com"
    在C:\Users\yum.ssh中打开id_ras.pub文件，复制文件中的内容到码云网站“安全设置”“SSH公钥”中，标题自定义

  7. 安装博客部署工具
    在创建的blog目录内输入以下代码
    npm install hexo-deployer-git --save
    cnpm install hexo-deployer-git --save

  8. 在码云网点击“管理”“功能设置”点击“初始化README启用SVN访问”

  9. 在码云网点击“服务”就可以点击“Gitee Pages”
    在”邮箱管理“中关掉”禁止命令推送暴露个人邮箱“。不关闭会报错

  10. 在C:\Users\yum\blog目录中打开_config.yml文件，
    在
    deploy:
    type: git
    下方（最后一行）粘贴以下内容
    repo: https://gitee.com/xxx/xxx.git
    branch: master
    将码云仓库首页“</>代码”，“克隆/下载”中复制HTTPS地址，粘贴到上面的repo后面
    theme: landscape 是主题的名字
    url: http://xxx.gitee.io/ url改为Gitee Pages的链接地址
    language: zh-CN 语言设置为中文
    title: 可以修改标题

  11. 部署博客

清理缓存文件db.json和已经生成的静态文件public
hexo clean

新建标题为XXXX的文章，这一步可以在md文件中编辑
hexo new "XXXX的文章"

生成网站静态文件到默认设置的public目录
hexo generate

启动本地博客服务器，预览主题效果
hexo server

自动生成静态文件，并部署到设定仓库
hexo deploy

注意事项

以后推送博文时重新上传步骤
hexo cl 清缓存
hexo g 重新生成文件
hexo d 上传
