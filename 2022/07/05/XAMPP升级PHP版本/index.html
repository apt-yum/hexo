XAMPP升级PHP版本

这种方式不用修改xampp配置文档内容就可以切换或者升级php版本：
以下方式适合php版本的小版本升级，如PHP Version 8.1.6升级为PHP Version 8.1.7；如果大版本升级，比如PHP Version 7.4升级为PHP Version 8.1.7以下方式并不适用（只能作为参考）；建议直接去官方网站https://www.apachefriends.org/download.html下载对映php版本的XAMMP，如在官网下载xampp-windows-x64-8.1.6-0-VS16-installer.exe安装包内所涉的环境为PHP Version 8.1.6，Apache/2.4.53 (Win64) OpenSSL/1.1.1n PHP/8.1.6 。

1、从官网https://windows.php.net/download/
下载php-8.1.7-Win32-vs16-x64.zip压缩包VS16 x64 Thread Safe (2022-Jun-07 22:31:20)；

2、将XAMPP安装目录D:\xampp\php重命名为php-8.1.6，这时在XAMPP控制面板中应该会报错，php服务无法启动；

3、将php-8.1.7-Win32-vs16-x64.zip压缩包解压到XAMPP安装目录D:\xampp\中，并重命名为php;

4、复制php-8.1.6目录中的文件及文件夹到php的目录中：
a. 把D:\xampp\php-8.1.6目录中的cfg，data，CompatInfo，docs，man，pear，scripts，tests，tmp，www，windowsXamppPhp几个文件夹及文件夹中所有文件复制到D:\xampp\php目录中；
b. 把D:\xampp\php-8.1.6\extras目录中除ssl目录以外的文件夹复制到D:\xampp\php目录中；
c. 把D:\xampp\php-8.1.6目录中的CompatInfo.php，pci，pci.bat，pci.css，pciconf，pciconf.bat，pear.bat，peardev.bat，pecl.bat，php.ini，phpunit，phpunit.bat，webdriver-test-example.php这些文件复制到D:\xampp\php目录中；

在XAMMP控制面板中启动apache服务，使用浏览器登录http://localhost，查询php版本号升级到Apache/2.4.53 (Win64) OpenSSL/1.1.1n PHP/8.1.7 。
